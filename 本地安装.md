## install python 3.5 on ubuntu 16.04 (Xenial)

```bash
# apt install
sudo apt-get install python3.5

# 设置默认python版本
sudo update-alternatives --install /usr/bin/python python /usr/bin/python3.5 1
sudo update-alternatives --config python # 选择python3.5对应的序号

# 安装pip
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
python get-pip.py

# 使用清华源加速python包安装
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

```

## install ROS Kinetic

按照http://wiki.ros.org/kinetic/Installation/Ubuntu安装说明执行以下步骤

```bash
# 1. setup sources.list
sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'

# 2. set up keys
sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-key 421C365BD9FF1F717815A3895523BAEEB01FA116

# 3. update and install
sudo apt update
sudo apt install -y ros-kinetic-desktop-full

# 4. initialize setup
sudo rosdep init

# if has error No module named 'rosdep2'
pip install rosdep

rosdep update

# 5. add env to bashrc
echo "source /opt/ros/kinetic/setup.bash" >> ~/.bashrc
source ~/.bashrc

# 6. install rosws
pip install rosinstall --user
```
